🔐 Linux User & Permission Management

👤 User Management

Create User

useradd username

Set Password

passwd username

Delete User

userdel username

👥 Group Management

Create Group

groupadd groupname

Add User to Group

usermod -aG groupname username

🔐 Permissions

View Permissions

ls -l

Change Permissions

chmod 755 file.sh

Change Ownership

chown user:group file.txt
